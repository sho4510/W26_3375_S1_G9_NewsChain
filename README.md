NewsChain

A news app that starts from the outcome and rewinds time to reveal the full story.

Course: CSIS-3375-001 | Section: S1 | Group: G9
Team Lead: Sho Ogawa (300411060)

Project Description
NewsChain is a narrative-driven mobile news application that helps users understand current events by starting from the present outcome and rewinding through past events to explain how and why the news happened.
Unlike traditional news apps that present isolated headlines, NewsChain displays news as a chain of connected events — each node shows a headline, source, date, and brief summary. Users navigate backwards through the chain to trace the full cause-and-effect story, from today's outcome all the way back to the origin event.
Key Features

Reverse-timeline story chain — Start from the latest news, tap back through history
Event count indicator — See how many events are in a chain before opening it (e.g. "4 events") and track your position within the chain (e.g. "Event 2 of 4")
Source credibility score — Each node displays a credibility rating so users can assess reliability
Chain Navigation onboarding screen — First-time users are guided through how the chain works
Reading Path — Bookmark chains and track reading progress
Explore screen — Search and filter by topic category
Profile & gamification — Streaks, achievements, and interest tags

App Motto
"Starting with the outcome, trace back to the cause."

Installation & Demo Instructions
This project is a Figma prototype (mobile app design). There is no code to install or run. To view the demo:
Option 1 — View in browser (recommended)

Open the Figma prototype link:
👉 https://www.figma.com/design/f8frhn1ZJTwjYEX1t53uni/
Click the ▶ Play button (top-right) to launch the interactive prototype
Use mouse clicks to navigate between screens as you would tap on a phone

Option 2 — View on mobile (Figma app)

Install the Figma app on your iOS or Android device
Log in to Figma and open the shared prototype link above
Tap the ▶ Play button to run the prototype in mobile view

How to interact with the prototype

Home screen → Tap a news card to open the story chain (Item Detail)
Item Detail → Tap backwards through chain nodes to trace the story
Home screen → Tap "How to interact" to see the Chain Navigation onboarding screen
Bottom nav bar → Navigate between Home, Explore, Reading Path, and Profile
Item Detail / News Detail → Tap "View Detail >" to see the Credibility screen
